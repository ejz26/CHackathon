This project is an interactive game created in C++ for a hackathon. The game was inspired by concepts from a research paper presented at a conference, and it applies those ideas in a playable, engaging format.

The goal of the project was to turn research-based concepts into an interactive experience where users can explore the main ideas through gameplay. The game includes user input, decision-making, and dynamic responses to create an engaging experience.

Features
Built using C++
Interactive gameplay with user input
Inspired by a conference research paper
Converts research concepts into a playable format
Created as part of a hackathon project
